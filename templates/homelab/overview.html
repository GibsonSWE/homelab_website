Overview

Having a homelab can be both educational and recreational. I like to describe my homelab as a miniature enterprise IT environment,
with many of the major components of a real enterprise infrastructure, but in a much smaller scale.

Hardware

My entire homelab currently runs on an Asus MiniPC.
This is the perfect form factor for a homeserver in my opinion, if you are living in an apartment or have limited space.
The server is also very quiet and is almost unnoticable unless I am running some CPU intensive tasks, which usually only happens for a limited time every now and then.

  • CPU: Intel Core i5 13500H (12 Cores, 16 Threads)
  • Memory: 32GB 4.8GHz DDR5
  • Storage: 2TB Kingston NV2 M.2 NVMe Gen 4
  • Ethernet: 1st 2.5Gb/s
  • Wi-Fi: Intel 802.11 AX
  • Bluetooth: Bluetooth 5
  • Dimensions (WxHxD): 120x58x130mm

OS/Hypervisor

The server is running Proxmox VE in a single node cluster. Proxmox is where I provision and manage all of my servers and their resources. Proxmox is also responsible for scheduled automatic backups of all of my servers, network firewalling, virtual network segmentation with VLAN tagging and interface configurations. For my servers running in Proxmox, I use a combination of virtual machines and containerization with LXC containers.

Network

My network so far is fairly simple. The physical topology consists of your typical home network with a single subnet. Inside the hypervisor network I am running OpenVSwitch and have segmented the network into several VLANs. Most of my servers have separate interfaces for management traffic and user traffic. My webservers are placed behind a reverseproxy that the user web traffic has to pass through.

Components

These are the services currently running in my homelab. You can read more about them in the components section.
I also have a few VMs running various versions of Linux and Windows for experimental purposes.

  • Pop!_OS Management VM
  • Bind9 DNS-server
  • NGINX Reverseproxy
  • Grafana Metrics server
  • Graylog Logging server
  • Netbox IP-address management server
  • Netprobe Internet quality monitoring tool
  • Home Assistant home automation platform
  • Cisco Modeling Labs Networking lab
  • Gibnet.se website webserver
